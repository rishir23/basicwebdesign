Review Cards: Important Terms

What is HTML?
HTML stands for HyperText Markup Language.
What is CSS?
CSS stands for Cascading Style Sheets. It styles HTML.
What does the <div> tag do?
It defines a container or section in HTML.
What is a CSS transition?
It creates smooth changes between CSS property values.
What is GitHub used for?
It hosts code and tracks changes using Git.
How do you apply a class in HTML?
Using class="classname" inside a tag.
What does hover mean in CSS?
It targets an element when the mouse is over it.
What does opacity control?
It controls how transparent an element is (0 = invisible, 1 = solid).
